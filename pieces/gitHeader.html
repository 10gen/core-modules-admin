GIT Documentation
GIT

The Git Local Manager is a tool to enable users to manage the local git repository as cloned on a given Environment. The GLM is intended to enable a user to pull the latest changes, commit changes made to files in the Environment, push committed changes to the central repository, and resolve conflicts.
